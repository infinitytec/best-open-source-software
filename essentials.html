Essential Open Source Software

Firefox

Firefox is simply the best web browser. It is built for speed, privacy, and pure awesomeness. Recent updates have made it significantly faster, more stable, and easier to customize. It is extremely cross-platform (with support for Windows, macOS, Linux, Android, and iOS).

Get it

uBlock Origin

uBlock origin is a browser extension that not only blocks ads, but anything else you want to hide. It is efficient with computer resources. It is available for Firefox, Chrome, Edge, Opera, and Safari.

Get it

LibreOffice

LibreOffice is a free, full-featured office suite. It has excellent compatability with Microsoft Office dcuments. It is really an amazing tool. It does what really expensive software does, but for free. It is available for Windows, macOS, and Linux. A file viewer is available for Android and a presentation remote is available for iOS and Android.

Get it

VLC

VLC is the best media player. It can handle (just about) any file you throw at it, which makes it useful for playing even the strangest file types. It can also play DVDs, and does so really well. It runs on every operating system and then some more. The interface can be a little clunky for some users, however the mobile apps are more streamlined for even the largest media libraries.

Get it

GIMP

GIMP is an alternative to Photoshop. It can do most of what Photoshop can do, and has some unnique features. The learning curve can be high, but once you get the hang of it it works great. It comes preloaded with a lot of tools and filters. It runs on Windows, macOS, and Linux.

Get it
